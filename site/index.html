Getting Started

1

Download

Grab the latest release for your platform from the Downloads section above.

2

Configure

Copy .env.example to .env and fill in your API tokens (PagerDuty, Jira, Databricks).

3

Run

Launch the toolkit and select the tool you need from the interactive menu. Each tool can also run standalone.
